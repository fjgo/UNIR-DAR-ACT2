Servicios

Los servicios en Angular son clases permiten encapsular la lógica de negocio y datos compartidos entre múltiples componentes. A continuación, aprenderemos cómo crear y usar servicios en Angular:

  1. Generar el servicio

    Usa Angular CLI para crear un servicio: 

    ng generate service nombre-servicio

    Esto creará dos archivos: nombre-servicio.service.ts y nombre-servicio.service.spec.ts (para pruebas).

    Un servicio típico se declara como sigue: 

    @Injectable({
  providedIn: 'root'
})
export class DataService {
  getData() {
    return ['dato1', 'dato2'];
  }
}

  2. Inyección de dependencias

    Para usar un servicio en un componente, debemos inyectarlo a través del constructor del componente. Por ejemplo: 

    import { NombreServicioService } from './nombre-servicio.service';

@Component({ ... })
export class NombreComponente {
    constructor(private servicio: NombreServicioService) {}
}

  3. Usar un servicio

    Define métodos en el servicio y accede a ellos desde el componente. Ejemplo básico: 

    export class NombreServicioService {
    getSaludo() {
        return 'Hola desde el servicio';
    }
}

export class NombreComponente {
    mensaje: string;
    constructor(private servicio: NombreServicioService) {
        this.mensaje = this.servicio.getSaludo();
    }
}
