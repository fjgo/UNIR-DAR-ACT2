Configuración del Proyecto

Configuración del Entorno

Antes de comenzar a trabajar con Angular, necesitamos configurar nuestro entorno de desarrollo. Para ello, debemos instalar Node.js y Angular CLI.

Instalación de Node.js

Dirígete a la página oficial de Node.js y descarga la versión recomendada para tu sistema operativo. En este enlace, podrás encontrar las versiones disponibles de Node.js, incluyendo las versiones LTS (Long Term Support) y las versiones actuales. Algunas de ellas pueden tener soporte a largo plazo, lo que te permitirá utilizarlas en producción sin problemas.

Existen unas versiones mínimas de node.js para poder trabajar con Angular.

Angular Node.js TypeScript RxJS
19.0.x ^18.19.1 / ^20.11.1 / ^22.0.0 >=5.5.0 / <5.7.0 / ^6.5.3 ^7.4.0

Una vez que hayas seleccionado la versión que deseas instalar, puedes descargarla y seguir las instrucciones para instalarla en tu sistema operativo.

Instalar Angular CLI

Abre una terminal y ejecuta el siguiente comando para instalar Angular CLI de manera global: 

npm install -g @angular/cli

Esto nos asegurará que la herramienta esté disponible en todo el sistema. Es importante asegurarte de tener Node.js instalado previamente, ya que Angular CLI depende de este entorno.

Crear un nuevo proyecto

Crear los archivos del proyecto según la estructura de Angular

Con Angular CLI instalado, puedes crear un nuevo proyecto ejecutando:

ng new tutorial-angular

Durante este paso, el sistema te preguntará si deseas incluir configuraciones como soporte para SCSS o configurar enrutamiento. Selecciona las opciones según tus necesidades. En nuestro caso seleccionaremos que queremos utilizar CSS para que Angular configure automáticamente los archivos.

Ejecutar el servidor de desarrollo

Una vez creado el proyecto, navega al directorio generado y levanta un servidor local con: 

cd tutorial-angular
ng serve

Este comando compilará la aplicación y la servirá mediante un servidor local, permitiéndote probar los cambios en tiempo real. Angular CLI también proporciona detección automática de cambios, por lo que cualquier modificación en el código se reflejará instantáneamente en el navegador.

Abrir la aplicación en el navegador

Angular CLI iniciará automáticamente el servidor en http://localhost:4200. Puedes acceder a la URL desde tu navegador para visualizar la aplicación inicial de Angular. Tambien puedes escribir o + enter para que el navegador abra directamente la página.

Esta aplicación incluye una plantilla básica que muestra un mensaje de bienvenida y sirve como punto de partida para el desarrollo.

Personalizar la configuración inicial

Una vez que el servidor está en funcionamiento, puedes comenzar a personalizar los archivos generados. Por ejemplo, modificando el archivo app.component.html para cambiar el contenido de la página principal. Tambien puedes agregar nuevas dependencias o librerías según las necesidades de tu proyecto utilizando npm install.

Esta aplicación inicial contiene una estructura básica que sirve como punto de partida para el desarrollo de tu aplicación.
