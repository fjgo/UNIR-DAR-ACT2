Directivas

Las directivas en Angular son instrucciones que modifican el comportamiento o la apariencia de elementos en el DOM. Existen tres tipos principales de directivas:

  1. Directivas estructurales

    Estas directivas alteran la estructura del DOM agregando, eliminando o manipulando elementos.

    • *ngIf: Se utiliza para mostrar o esconder un elemento según una condición booleana. 

      <div *ngIf="condicion">Este texto se muestra si la condición es verdadera.</div>

    • *ngFor: Repite un elemento por cada ítem en una colección. 

      <ul>
  <li *ngFor="let item of lista"> {{'{{ item }}'}}</li>
</ul>

  2. Directivas de atributos

    Estas directivas modifican el comportamiento o estilo de un elemento existente.

    • ngClass: Permite agregar clases dinámicamente a un elemento según una condición. 

      <div [ngClass]="{ 'clase-activa': isActive, 'clase-inactiva': !isActive }">
  Contenido con clases dinámicas.
</div>

    • ngStyle: Se usa para aplicar estilos dinámicos. 

      <div [ngStyle]="{ 'color': colorTexto, 'font-size': tamanioFuente }">
  Texto con estilos dinámicos.
</div>

  3. Directivas personalizadas

    Angular permite crear directivas propias para extender la funcionalidad del DOM. Para crear una directiva personalizada:

    • Generar una directiva con Angular CLI: 

      ng generate directive nombre-directiva

    • Implementar la lógica: Define la funcionalidad en el archivo TypeScript. 

      import { Directive, ElementRef, Renderer2, HostListener } from '@angular/core';

@Directive({
  selector: '[appResaltar]'
})
export class ResaltarDirective {
  constructor(private el: ElementRef, private renderer: Renderer2) {}

  @HostListener('mouseenter') onMouseEnter() {
    this.renderer.setStyle(this.el.nativeElement, 'background-color', 'yellow');
  }

  @HostListener('mouseleave') onMouseLeave() {
    this.renderer.removeStyle(this.el.nativeElement, 'background-color');
  }
}

    • Uso de la directiva: Aplica la directiva a un elemento HTML usando su selector. 

      <p appResaltar>Este texto se resaltará al pasar el ratón por encima.</p>

Con estas directivas, podemos mejorar considerablemente la interacción y el dinamismo de nuestas aplicaciones Angular, haciendo que el código sea más limpio y reutilizable.
