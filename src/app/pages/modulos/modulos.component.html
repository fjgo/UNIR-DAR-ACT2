Módulos

Los módulos en Angular son contenedores para agrupar componentes, directivas, servicios y otros módulos. El módulo principal, AppModule, es donde Angular busca el punto de entrada de la aplicación. Puedes crear módulos adicionales para organizar el código según funcionalidades específicas. Por ejemplo, un módulo para gestionar usuarios o un módulo para mostrar informes.

Para crear un módulo con Angular CLI:

ng generate module nombre-modulo

El archivo generado incluirá configuraciones básicas como @NgModule y un arreglo para importar y declarar componentes asociados al módulo. Usar módulos ayuda a cargar funcionalidades de manera diferida (lazy loading) para mejorar el rendimiento de la aplicación.

Un ejemplo de configuración básica sería:

@NgModule({
declarations: [Componente1, Componente2],
imports: [CommonModule],
exports: [Componente1]
})
export class NombreModulo {}

Los módulos tambien permiten segmentar el código de manera lógica, facilitando su mantenimiento. Por ejemplo, si estás desarrollando una tienda en línea, podrías tener módulos separados para la gestión de productos, usuarios y órdenes.
